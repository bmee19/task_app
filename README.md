# TaskApp( 欠品タスク管理アプリ）

## 概要
欠品対応を効率化するために、SwiftUIとFirebaseを用いて個人開発したiOS用の情報伝達アプリです。
欠品タスクの登録、タスクのステータス管理(対応中・完了)、店内マップ上での位置登録/表示、履歴管理、全体チャットを備えています。
